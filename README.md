# 🖥️ Simple GUI Calculator (Python Tkinter)

## 📌 Overview

This project is a basic graphical calculator built using Python's Tkinter library.
Unlike command-line programs, this version provides a simple window-based interface where users can perform calculations by clicking buttons.

It’s a beginner-friendly project that demonstrates how to build interactive desktop applications in Python.

---

## 🚀 Features

* Perform basic arithmetic operations (+, -, ×, ÷)
* Modulus (%) support
* Button-based input system
* Real-time calculation display
* Error handling for invalid expressions
* Clean and minimal user interface

---

## 🛠️ Built With

* Python
* Tkinter (built-in GUI library)

---

## ▶️ How to Run

1. Make sure Python is installed on your system
2. Copy the code into a file named `main.py`
3. Run the program using:

```bash
python main.py
```

A calculator window will open on your screen.

---

## 🧠 How It Works

* The interface is created using Tkinter widgets like `Entry`, `Button`, and `Frame`
* User input is collected through button clicks
* The `eval()` function is used to evaluate mathematical expressions
* Errors (like invalid input or division issues) are handled gracefully
