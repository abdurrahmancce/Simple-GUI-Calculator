# 🖥️ Simple GUI Calculator (Python Tkinter)

## 📌 Overview

This project is a graphical calculator built using Python and Tkinter.
It started as a simple calculator and was later enhanced with additional features to improve usability and functionality.

---

## 🚀 Features

### 🔢 Core Features

* Basic arithmetic operations (+, -, ×, ÷)
* Modulus (%) support
* Button-based input system
* Real-time result display
* Error handling for invalid expressions

---

### ⌨️ Keyboard Input Support

* Users can type directly using the keyboard
* Supports numbers and operators
* Improves speed and usability

---

### 🌙 Improved UI Design

* Cleaner and more structured layout
* Better spacing and alignment
* Ready for dark mode customization
* More user-friendly interface

---

### 🕘 Calculation History

* Stores previous calculations
* Displays past results
* Helps track user operations

---

### 🧪 Scientific Calculator (Extended Version)

* Power operations (x², xʸ)
* Additional mathematical functions (future scope)
* Designed for advanced calculations

---

## 🛠️ Built With

* Python
* Tkinter

---

## ▶️ How to Run

1. Install Python
2. Save the file as `main.py`
3. Run:

```bash
python main.py
```

---

## 🧠 How It Works

* Tkinter is used to build the GUI interface
* Button clicks and keyboard inputs update the display
* Expressions are evaluated dynamically
* History is stored in a list structure

---

## 📚 Learning Outcomes

* GUI development with Tkinter
* Event handling (button + keyboard)
* Improving UI/UX design
* Managing state (calculation history)
* Structuring scalable Python projects
